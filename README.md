# Relatórios Diários

Sistema web desenvolvido para registrar e gerenciar atividades diárias, permitindo a geração automática de relatórios organizados e profissionais.

## Sobre o sistema

O sistema permite registrar atividades realizadas ao longo do dia de forma rápida e prática.

Ao criar uma nova atividade, o usuário pode informar manualmente o horário de início ou utilizar o horário atual sugerido pelo sistema. Caso a atividade permaneça em andamento, ela poderá ser encerrada posteriormente, momento em que o sistema preencherá automaticamente o horário de término. Também é possível editar esse horário manualmente, caso o encerramento não tenha sido registrado no momento correto.

### Funcionalidades

* Cadastro de múltiplas atividades por dia.
* Edição de atividades já registradas.
* Definição manual dos horários de início e término.
* Controle de atividades em andamento.
* Navegação entre registros de diferentes datas.
* Geração automática do relatório diário.
* Cópia do relatório para a área de transferência.
* Exportação do relatório em PDF.
* Sistema de autenticação de usuários.

### Objetivo

Centralizar o registro das atividades diárias e facilitar a criação de relatórios organizados, reduzindo o tempo gasto com documentação manual e aumentando a precisão das informações registradas.
